About This Project

A smart solution for handling and converting quantities across multiple units efficiently

Project Overview

This is a full-stack web application that performs quantity measurements and conversions across multiple unit types. It features a Spring Boot REST API backend with MySQL database and a vanilla JavaScript frontend with modern UI design.

The application supports both anonymous and authenticated users, providing instant calculations for everyone while saving calculation history for registered users.

Purpose of the Project

This project is developed as part of the BridgeLabz Corporate GenAI Fellowship Program to demonstrate full-stack development skills, backend architecture design, and modern web development practices.

The application showcases enterprise-level features including JWT authentication, OAuth2 integration, RESTful API design, and responsive UI development using industry best practices.

Key Features

  • Anonymous calculations - No registration required
  • Compare quantities across different units (LENGTH, WEIGHT, VOLUME, TEMPERATURE)
  • Convert between measurement systems with high precision
  • Arithmetic operations: Add, Subtract, Divide quantities
  • User authentication with JWT and Google OAuth2
  • Calculation history tracking for registered users
  • User profile management and customization
  • Responsive design with dark/light theme support
  • Enterprise-grade Spring Boot backend with MySQL
  • RESTful API architecture with proper error handling
